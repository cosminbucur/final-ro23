# shortcuts

## code
- duplicate code        `CTRL + ALT + SHIFT + NUMPAD 2`
- find text in project  `CTRL + SHIFT + F`
- find text in file     `CTRL + F`
- find file in project  `CTRL + P`

- intellisense          `CTRL + .`
- multi cursor          `CTRL + SHIFT + arrow`

## terminal
- open terminal         `CTRL + ~`
- start server          `ng serve`
- stop server           `CTRL + C` > yes
